# Lista

A local-first task list/shopping list with conflict-free multiplayer editing and offline support.

Made using [SvelteKit](https://svelte.dev/) and [Automerge](https://automerge.org/), hosted on [lista.matsku.dev](https://lista.matsku.dev).
